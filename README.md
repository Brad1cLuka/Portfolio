Projects
1. Clinic VILM (Team Project)
Description:
Clinic VILM is a desktop application designed for healthcare clinics and patients. It supports role-based logins, allowing administrators, doctors, and patients to access different functionalities. Key features include adding new patients, managing healthcare staff through the administrator panel, prescription writing, patient medical history tracking, electronic health records, appointment scheduling with doctors, and an integrated map of the clinic.

Teamwork:
This project was developed collaboratively with Vladimir and Ilija. All team members contributed equally, and most forms and functionalities share similar design and logic since we worked together on them throughout the project.

Technologies: C#, .NET, Windows Forms (or specify if you used WPF/WinForms)
